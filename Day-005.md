## Commands Practiced
| Command | What I Now Understand About It |
|---------|-------------------------------|
| `head -n <no.> <file name>` | Prints first n no. of lines of a file |
| `tail -n <no.> <file name>` | Prints last n no. of lines of a file |
| `touch <file name>` | For creating a new empty file |
